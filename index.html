Crystal Match Game

This is how you play the game!

  1. First when you refresh the page a random number is shown
  2. Then each crystal will have a random value from 2 to 12
  3. Try and match your score with computer score by selecting the crystals, each click will add that value to your total score number
  4. Your wins and losses will be set on the page also you can reset the game by refreshing the page
